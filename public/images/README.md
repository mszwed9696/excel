# Images Directory

## Required Logo Files

Please add the following logo file to this directory:

1. **logo.png** - Main logo file (the Excel Real Estate logo - black diamond with gold house)
   - Recommended size: 512x512px or larger (for best quality when generating favicons)
   - Format: PNG with transparent background preferred
   - This file is used in the header and footer

## How to Add the Logo

1. Save your Excel Real Estate logo as `logo.png` in `/public/images/`
2. The site will automatically use this in the header and footer

## Logo References

The logo is referenced in:
- Header component (`/components/Header.tsx`) - 48x48px display
- Footer component (`/components/Footer.tsx`) - 40x40px display
